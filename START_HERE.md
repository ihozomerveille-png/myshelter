## 🚀 Quick Start (3 Steps)

### 1. Backend Setup
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with MongoDB URI
npm run dev
```

### 2. Frontend Setup (new terminal)
```bash
cd frontend
npm install
npm start
```

### 3. Access
- **Frontend:** http://localhost:3000
- **Backend:** http://localhost:5000
